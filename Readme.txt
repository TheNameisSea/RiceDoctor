1. Phần mềm
AI xác định bệnh của lúa dựa trên hình ảnh

2. Cách sử dụng
Để sử dụng app, phải cài đặt sẵn ngôn ngữ lập trình python, phần mềm anaconda và các thư viện: sklearn, pandas, numpy, pillow, opencv, albumentations, pytorch, timm, streamlit
Cách dùng app:
1. Mở 1 terminal shell bất kỳ, có thể là powershell hoặc bash
2. Sử dụng shell để đến folder chứa file phần mềm
3. gõ lệnh "streamlit run app_copy.py"

folder test_img trong folder input là folder chứa các file ảnh để test
